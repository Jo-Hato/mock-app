MockApp (JP)

この実験の所要時間は約7〜12分です。


本研究の目的は、事前知識バイアスを避けるため、この実験の最後に開示されます。参加者は本研究の目的を知った上で、データを送るかどうかを選択することができます。


この実験を始める前に、 ブラウザをGoogle Chromeにしてください。LINE等のアプリから開かれるブラウザは使わないでください

また、できれば椅子に座り、携帯電話は縦向きで、手にとってご使用ください。




自動補正を全てOFFにした後、 "Allow Sensor Use"ボタンを押し、"Next"ボタンを押してください

また、以下のボタンを押すと、上記のプライバシーポリシーに同意して頂いた事とします。






    Debugger:
  • eventNum = {{ eventNum }}
  • touchNum: {{ touchNum }}
  • delNum: {{ delNum }}
  • runData:
    • subjectInfo: {{ runData.subjectInfo }}
    • groundTruth1: {{ runData.groundTruth1 }}
    • groundTruth2: {{ runData.groundTruth2 }}
    • record1: {{ runData.record1 }}
    • record2: {{ runData.record2 }}
